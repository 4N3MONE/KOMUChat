# KOMUChat : 인공지능 학습을 위한 온라인 커뮤니티 대화 데이터셋

## 개요
에펨코리아 + 인스티즈에서 수집한 약 3만건의 문장 페어 데이터

## 속성 정보
Q : 커뮤니티 게시글의 제목을 활용한 질문 문장

A : 게시글에 달린 첫 번째 댓글을 활용한 답변 문장

tag : 커뮤니티 게시판 내 세부 태그/탭

src : 데이터 출처 사이트

## 데이터 예시
| Q | A | tag | src |
|-----------|-----------|-----------|-----------|
| 셀 1      | 셀 2      | 셀 3      | 셀 4      |
| 셀 5      | 셀 6      | 셀 7      | 셀 8      |
| 셀 9      | 셀 10     | 셀 11     | 셀 12     |
